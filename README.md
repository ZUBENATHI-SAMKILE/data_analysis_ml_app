## Data Analysis ML App

- A full-stack platform for data analytics, machine learning predictions, and interactive dashboards, supporting multiple datasets including COVID-19 Trends, HR Analytics, and Sales Forecasting.

- This project allows users to upload datasets, visualize insights, and generate machine learning predictions through an easy-to-use web interface.

## Live Demo

 https://zubenathi-samkile.github.io/data_analysis_ml_app/

## What the App Does

 1. Data Upload & Processing

- Users can upload CSV/Excel datasets for analysis.
- The backend processes data using Python (Pandas, NumPy, Scikit-Learn).

 2. Interactive Data Visualization

- Visual charts generated including:
- Line charts
- Bar charts
- Pie charts
- Heatmaps
- Trend analysis

 3. Machine Learning Predictions

- Prebuilt ML models support predictions for:
- COVID-19 Future Cases
- Employee Attrition (HR Analytics)
- Sales Forecasting
- Productivity Metrics

 4. REST API (Backend)

- The backend provides endpoints for:
- Data processing
- Model predictions
- Dashboard summaries
- Dataset validation

5. Responsive React Frontend
